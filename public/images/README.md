# Obrázky

## Struktura

### Placeholder obrázky nemovitostí
Umístěte do `placeholders/`:
- `property-1.jpg` až `property-6.jpg`
- Rozlišení: minimálně 800x600px
- Formát: JPG nebo WebP
- Obsah: luxusní, neutrální stock fotky nemovitostí u moře

### Hero obrázky destinací
Umístěte do `hero/`:
- `spanelsko.jpg`
- `chorvatsko.jpg`
- `italie.jpg`
- `recko.jpg`
- `portugalsko.jpg`
- `francie.jpg`
- `turecko.jpg`
- `bulharsko.jpg`
- `cyprus.jpg`
- `malta.jpg`
- Rozlišení: minimálně 1200x600px
- Formát: JPG nebo WebP

### Open Graph obrázek
- `og-default.jpg`
- Rozlišení: 1200x630px (doporučeno)
- Formát: JPG nebo PNG

## Poznámky

- Všechny obrázky by měly být optimalizované pro web
- Použijte alt texty podle specifikace v komponentách
- Placeholder obrázky jsou rotovány podle indexu
